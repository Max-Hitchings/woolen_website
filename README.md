# Woolen Website

## Overview
Woolen is a premium wellness brand site for a launch-stage merino wool sauna hat. The current product experience is a content-led landing page plus supporting brand and policy pages, designed to build trust and capture waitlist signups.

## Stack
- Next.js 16 App Router
- React 19
- TypeScript
- Tailwind CSS 4
- Framer Motion
- Vercel deployment target

## Commands
- `npm run dev` - start the local dev server
- `npm run build` - build for production
- `npm run lint` - run ESLint
- `npm run start` - run the production server after a build
